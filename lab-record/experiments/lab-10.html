Lab 10: Full-Stack Application

1. Question

Create a http server. Use the url module to parse the incoming request URL and extract the pathname for routing decisions. Read HTML files from the current directory.

Implement these routes:

  • For pathname /, read and serve the content of index.html. If the file doesn't exist, log an error in a log file but still return a basic "Home Page" message.
  • For pathname /about, read and serve the content of about.html similarly and log an error in a log file case it is not found.
  • For any other pathname, return a 404 status with a Page Not Found.

Create two simple HTML files in the same folder: index.html (with a title and a paragraph) and about.html (with a title and a paragraph).

Compress the project folder and upload it to the submission portal. Push the code to a public GitHub repository, host the application online on free hosting services, ensuring it matches local functionality, and provide the live URL in a README.md file.
