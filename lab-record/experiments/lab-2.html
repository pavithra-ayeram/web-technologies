Lab 2: Weather Dashboard

1. Question

Develop a weather dashboard that fetches and displays current weather data for a city using a free public API.

  • Requirement Analysis: Define the essential data points to display (e.g., temperature, city name, weather icon, etc).
  • Design: Sketch a "weather card" UI to present the information. Use Figma or any drawing/designing tool of your choice. Modularize your plan.
  • Implementation: Build, test, and integrate module by module. Use HTML for structure, CSS for styling, and JavaScript fetch API (with async/await) to handle the API request and update the DOM.
  • Testing: Test for successful data fetching and display, as well as error handling for invalid city names. Identify other test scenarios.
  • Deployment: Manage your project in a GitHub repository with a proper README file and deploy it live using GitHub Pages or other free hosting services.
