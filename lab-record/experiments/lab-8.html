Lab 8: Social Media Dashboard

1. Question

Build a simple React application using React Bootstrap to create a blog dashboard with the following features. Create a main dashboard that includes:

  • A responsive Navbar (using React Bootstrap's Navbar component) with links to "Home" and "Posts".
  • A grid of 3-4 blog post cards (using Card and Grid/Row/Col components). Each card should show a post title, short description, and a "Read More" button (using Button component).
  • Use functional components and props to pass post data (e.g., title and description) from a parent component to the Card components.
  • Draw the component hierarchy and use functional components, props, useState, and useEffect for managing timer intervals.
  • Compress only the files in the src folder into a zip file, upload, and submit them.
